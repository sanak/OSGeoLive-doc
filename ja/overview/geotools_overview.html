GeoTools¶

GeoTools は Open Geospatial Consortium (OGC) の標準に準拠した空間データの操作やデータ構造を利用するための、オープンソース (LGPL) の Java ライブラリです。

GeoTools は Web サービスやコマンドライン、デスクトップアプリケーションなどを含むいくつかのプロジェクトで利用されています。OSGeoLive のアプリケーションで GeoTools を利用しているものは以下となります: 52 North SOS, 52 North WPS, AtlasStyler, Geomajas, Geopublisher, GeoServer, 及び uDig

主要機能¶

  • 主要な空間概念とデータ構造のためのインタフェースの定義

    • 統合されたジオメトリのサポートは、JTS Topology Suite(JTS) によって提供されます。
    • OGC Filter Encoding仕様を使用する属性および空間フィルタ

  • スレッド間、トランザクションサポート、スレッド間のロックをサポートするクリーンなデータアクセスAPI

    • 多くのファイル形式および空間データベースのGISデータにアクセスする
    • 座標参照系と変換サポート
    • 広範囲のマップ投影を使用する
    • 空間的および非空間的属性に関してデータをフィルタ処理および分析する

  • ステートレスでメモリの少ないレンダラ。サーバ側の環境で特に便利です。

    • 複雑なスタイルを使用してマップを作成および表示する

  • XML Schemaを使用してGMLコンテンツにバインドする強力な*schema assisted*解析テクノロジ

    • 構文解析/符号化技術は、GML、Filter、KML、SLD、およびSEを含む多くのOGC標準のバインディングを備えています。

  • GeoToolsプラグイン:ライブラリに追加のフォーマットを追加ができるオープンなプラグインシステム

    • GDALから追加のラスター形式を読み込めるようにするImageIO-EXTプロジェクトのプラグイン

  • GeoTools拡張機能

    • コアライブラリのspatial facilitesを使用して構築された追加機能を提供します。
    • 拡張機能は、(最短パスを見つけるための)グラフとネットワーキングのサポート、検証、Webマップサーバクライアント、XML構文解析とエンコーディングのバインディング、そしてカラーbrewerを提供します。

  • サポートされていないGeoTools

    • GeoToolsはまた、新しい人材を育成し、実験を促進するために使用されるステージング領域を持つ、より広いコミュニティの一部として動作します。
    • 注目すべき点は、スイングサポート(geotoolsチュートリアルで使用)、swt、ローカルおよびWebプロセスサポート、追加の記号、追加のデータフォーマット、グリッドの生成、およびISOジオメトリのいくつかの実装です。

サポートするフォーマット¶

  • ラスター形式とデータアクセス:

    arcsde, arcgrid, geotiff, grassraster, gtopo30, image (JPEG, TIFF, GIF, PNG), imageio-ext-gdal, imagemoasaic, imagepyramid, JP2K, matlab.

  • データベース 「jdbc-ng」のサポート:

    db2, h2, mysql, oracle, postgis, spatialite, sqlserver.

  • ベクタフォーマットとデータアクセス:

    app-schema, arcsde, csv, dxf, edigeo, excel, geojson, org, property, shapefile, wfs.

  • XML バインディング:

    次データのために提供されるJavaデータ構造とバインディング: xsd-core (xml simple types), fes, filter, gml2, gml3, kml, ows, sld, wcs, wfs, wms, wps, vpf.

    DOMおよびSAXサプリケーションで利用できる追加のジオメトリ、フィルタ、スタイルのパーサ/エンコーダ。

実装されている標準規格¶

多数のオープン地理空間コンソーシアム(OGC)標準のサポート:

  • OGC Style Layer Descriptor/記号データ構造のエンコードとレンダリングエンジン
  • Simple Featureサポートを含むOGC General Feature Model
  • ラスタ情報表現のOGC Grid Coverage
  • OGC Filterと Common Constraint Language (CQL)
  • Web Feature Service(WFS)、Web Map Service(WMS)、およびWeb Process Service(WPS)の試験的サポートのクライアント
  • ISO19107ジオメトリ
