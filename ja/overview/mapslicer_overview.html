MapSlicer¶

Proxy WMS & タイル サービス¶

マップ・パブリッシング¶

Mapスライサーは、ラスター・マップ・パブリッシングのためのマップ・タイルを作成するためのデスクトップ・アプリケーションです。Geodata は、 Google マップや地球と互換性のあるタイルに変換されて、直接ウェブサーバやクラウドストレージ (Amazon S3など) へアップロードされることで、公開できるようになります。

サーバー・サイドでの大規模な構成変更は必要ありません。単純なファイルのホスティングはとても簡単です。強力なブラウザ機能により、パンやズーム、マーカーやベクタデータのオーバーレイなどの動的な操作が可能です。

アプリケーションは、 OpenLayers や Google マップ API をベースにしたシンプルなビューアーをすぐに生成し、簡単にカスタマイズできる。

主な機能¶

  • 既知の Javascript API サポート: OpenLayers、 Google マップ API
  • 追加のサーバー・ソフトウェアのインストールは不要です
  • ほとんどあらゆる場所でホスティング : Amazon S3 などのクラウド・ストレージ、または FTP アクセスによる安価の 「無制限の」 Web ホスティング
  • 商用レイヤ (Google、 Bing、 Yahoo! ) または OpenStreetMap を使用した簡単なマッシュアップ
  • OSGEO TMS (タイル・マップ・サービス仕様) に従います。
  • さまざまなフォーマットでラスターデータを処理する機能 : TIFF/GeoTIFF、 MrSID、 ECW、 JPEG2000、エルダス HFA、 NOAA BSB、 JPEG など。
