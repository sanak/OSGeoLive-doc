Asenna OSGeoLive kovalevyllesi¶

OSGeoLive voidaan asentaa kovalevyllesi samalla tavalla kuin Ubuntu jakelu on asennettu.

Tämä prosessi vaidaan suorittaa Virtuaali koneessa jotta luodaan pysyvä OSGeoLive Virtuaali kone DVDltä tai ISO Imagesta.

Järjestelmävaatimukset¶

  • 20 GB vapaata kovalevytilaa

Asenna OSGeoLive¶

Valitse Install RELEASE

Anna salasana=”user”

Valitse kieli

Valitse aikavyöhyke

Valitse näppäimistö

Specify which disk to want to install on. (Beware, this may delete an existing installed partition).

Oletusarvot asennukselle ovat:

  • Nimi = ”user”
  • käyttäjänimi = ”user”
  • salasana = ”user”
  • tietokoneen nimi = ”osgeolive”
  • Sisäänkirjoitus automaattisesti

Valitse Install

Mene kahville sillä välin kun asennusta suoritetaan.

Siinä kaikki.

Vihje

Installation Tips In case you are installing using a username other than ”user”, tomcat applications require to have permission to service command as described in https://trac.osgeo.org/osgeo/ticket/1153
