Geomajas¶

Böngésző GIS kliens¶

A kliens lehetővé teszi a fejlesztőknek, hogy hatékony web GIS alkalmazást hozzanak létre, interaktív térképekkel, kibővíthető vezérlőkkel és egyedi felhasználási esetekkel. Az adatforrások és funkciók széles köréhez biztosít hozzáférést a böngésző programon keresztül.

Alapfunkciók¶

  • A GWT-n keresztül tisztán Javaban programozható
  • Sok réteg típust támogat (WMS, WFS, WMTS, TMS, OSM, Google Maps, Shape, PostGIS, Oracle, JDBC, …)
  • Kézreálló beépített vezérlők (szerkesztés, piros vonalazás, nyomtatás, geokódolás, réteg fa, …)
  • Szabja testre a saját alkalmazási igényeihez és hozzon létre új vezérlőket
  • Könnyen átalakítható egyéni CSS-en keresztül
  • Nincs szükség bögészőbe beépülő modulokra
  • Az internet eszközök széles skáláján fut HTML5 felhasználásával
  • Nem egy GWT rajongó? Érje el az alapfunkciókat a JavaScript API-n keresztül.

Vállalati szerver tulajdonságok¶

  • Fejlett biztonság különböző szinteken (réteg, elem, attribútum, funkcionalitás, …)
  • Zökkenőmentes integráció az ön saját domain modelljével - perzisztencia a Hibernate, web szolgáltatások Java RMI, … alkalmazásokon keresztül.
  • Vállalati integráció más üzleti alkalmazásokkal, mint például JBPM, Activiti, JasperReports, CMS, Alfresco, ipari szabványokon keresztül (SOAP, REST, ESB, JNDI, …)
  • Könnyen kibővíthető a Spring keretrendszer segítségével.
