Mapnik Quickstart¶

Mapnik ist ein Werkzeug, um Kartenanwendungen zu entwickeln. Mapnik wird meist dazu genutzt, Karten zu generieren. Mapnik ist leicht erweiterbar und sowohl für Web- als auch Desktop-Entwicklung geeignet.

Mapnik & Python¶

Mapnik und die dazugehörigen Python-Abhängigkeiten sind auf dem aktuellen System installiert und können über entsprechende Scripts angesprochen werden.

Das Erstellen von Karten in Python ist mit Mapnik ganz einfach. Öffnen Sie ein Terminalfenster, und geben Sie ‚python‘ in der Befehlszeile ein, um einen Python-Interpreter zu öffnen. Probieren Sie dann:

>>> import mapnik, os
>>> m = mapnik.Map(600,400)
>>> style = '/usr/local/share/mapnik/demo/population.xml'
>>> mapnik.load_map(m,style)
>>> m.zoom_all()
>>> mapnik.render_to_file(m,'map.png')
>>> os.system('xdg-open map.png')

Bemerkung

Der oben dargestellte Code beruht darauf, dass ein XML Stylesheet an Mapnik übergeben wird. Hieraus werden die Layerdaten und die dazugehörigen Styles gelesen. Diese können zum Beispiel innerhalb von QGIS mit Quantumnik erstellt werden: https://plugins.qgis.org/plugins/quantumnik/

Mapnik & Leaflet¶

Es steht eine grundlegende Demoanwendung zur Verfügung, die die Verwendung von Mapnik verdeutlicht und nutzt, um Kacheln im OSM/Google-Tile-Schema (Kachel-Schema) bereitzustellen und in einer OpenLayers-Anwendung anzuzeigen.

Die Demo verwendet einen Kachelserver namens „TileStache“, der für Mapnik entwickelt wurde. Besuchen Sie die TileStache Webseite für weitere Informationen: http://tilestache.org/

  1. Klicken Sie auf :menuselection:‘Desktop –> Spatial Tools –> Start Mapnik & TileStache‘. Der Server startet in einem Terminalfenster (das Fenster bleibt geöffnet und gibt grundlegende Debuginformationen aus).
  2. Überprüfen Sie, ob der Server funktioniert, indem Sie eine Kachel vom Server anfordern. Klicken Sie auf diesen Link http://localhost:8012/example/0/0/0.png und verfolgen Sie die Ausgabe im Server-Terminalfenster.
  3. Besuchen Sie die Leaflet-Demo-Anwendung unter ‚http://localhost/leaflet-demo.html <.. ./../leaflet-demo.html=““>‘_.</..>
