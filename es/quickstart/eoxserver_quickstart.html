Inicio Rápido de EOxServer¶

EOxServer is a server for presenting big Earth Observation (EO) data archives and metadata via Open Standards. It builds upon an Open Source stack of software which includes Python, MapServer, Django/GeoDjango, GDAL, PROJ and a SpatiaLite or PostGIS database.

Esta Guía de Inicio Rápido describe cómo:

  • Utilice el Cliente Web para ver, filtrar, crear subconjuntos y descargar datos de EO
  • Utilice el Cliente de Administración para inspeccionar los archivos de datos de EO

Iniciar EoxServer¶

Elija Geospatial ‣ Web Services ‣ EOxServer. Esto lanza un explorador que muestra la instancia de demostración disponible en http://localhost/eoxserver/

Trabajar con el Cliente Web¶

Click en el vínculo Cliente Web para abrir el cliente EOxServer integrado.

Dentro de este cliente puede explorar el contenido de la instancia de EOxServer. La instancia de demostración está llena de escenas de ENVISAT MERIS.

El cliente proporciona un conjunto de interacciones. En la parte superior se puede ver una barra de menús para mostrar la Selección de Capas y las Herramientas widgets . El área principal es el widget de mapa del cliente con un conjunto de conjuntos de datos ya desplegados. En la parte inferior, encontrarás el widget timeslider, que te permite explorar el contenido en la dimensión temporal.

The Layer Selection widget allows you to enable or disable the visibility and configure the rendering of the layers. Also, you can show or hide the streets-overlay and select the background layer. Please note, that when you disable the «MER FRS 1P RGB reduced» layer, the timeslider widget is hidden and only reappears once a dataset layer is enabled. The Tools widget provides three different tools: the Bounding Box Selection Tool, the Selection Management Tool and the Download Tool.

De manera similar a un mapa, puede hacer zoom o mover el actual tiempo de interés visible y seleccionar un tiempo de interés dibujando en el área que contiene los puntos rojos. Tambén puede mover el cursor sobre un punto para ver el conjunto de datos ID. Haciendo click en el punto, el mapa se acerca automáticamente a la extensión del conjunto de datos.

Cuando se habilita la Herramienta de Recuadro Delimitador, puede dibujar un recuadro delimitador directamente en el mapa. Ese recuadro se utiliza para realizar consultas dentro de la Herramienta de Descarga.

When the Download Tool is enabled, a query to the server is sent. This query includes the selected time of interest and bounding box. The results are shown in the Download widget, where you can select the format, projection and datasets to download.

Esto fue una rápida introducción al Cliente Web EOxServer. Por favor consulte la documentación en línea para obtener más información sobre este tema.

Trabajar con el Cliente de Administración¶

Desde la página principal de EOxServer, haga click en el enlace Cliente Administrador e inicie sesión con el usuario admin y la contraseña admin.

El Cliente de Administración es administrador estándar de Djangos y le permite configurar los datos disponibles. Por favor, siéntase libre de explorar al cliente. Puede encontrar más información en operators” guide.

Por ejemplo, puede inspeccionar la serie de conjuntos de datos que se carga de forma predeterminada haciendo click en el vínculo «Serie de Conjuntos de Datos» y, a continuación, en el vínculo «MER_FRS_1P_RGB_reduced».

Aquí también puede ver los conjuntos de datos incluidos. Puede excluir conjuntos de datos individuales utilizando la casilla de verificación «eliminar» a la derecha y hacer click en «Guardar».

  • Si usted ya está en el entorno de OSGeoLive, lea la copia local de la Documentación Local de los Usuarios de EOxServer
