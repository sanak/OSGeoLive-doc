Guía de Inicio Rápido de OpenStreetMap¶

OpenStreetMap es un mapa libre de nivel calle del mundo, creado por una comunidad permanentemente creciente de mappers. Cualquiera puede editar OpenStreetMap. El Sitio Web Aprenda OSM proporciona guías fáciles de entender, paso a paso, para que usted pueda comenzar a contribuir a OpenStreetMap y usando los datos OpenStreetMap . La guía Aprenda OSM en el enlace de arriba paso a paso muestra cómo navegar por el Sitio Web de OpenStreetMap, vistas y mapas de impresión y regístrese para una cuenta de usuario. Después de tener tu propio nombre de usuario y contraseña, podrás añadir tus primeros puntos en el mapa, aprende a hacer encuestas de calle y trabajo con imágenes aéreas.

El resto de esta guía presenta parte del software en línea y fuera de línea utilizada para editar y representar datos OSM.

Programas que funcionan con datos OSM¶

iD¶

** iD ** es el editor en línea de predeterminado para OpenStreetMap (OSM). iD es rápido y fácil de usar y permite la asignación de datos de fuentes como la encuesta de terreno, vía satélite e imaginarios aéreas, GPS rastros o imágenes a escala vial.

  • Página de Inicio: https://www.openstreetmap.org/edit
  • Documentación: https://wiki.openstreetmap.org/wiki/ID
  • Tutorial: https://learnosm.org/en/beginner/id-editor/

JOSM¶

JOSM (Java OpenStreetMap Editor) es un avanzado editor para OpenStreetMap (OSM), escrito en Java. Es más potente y complejo que iD, puede utilizarse para la edición masiva, fuera de línea o en una conexión a Internet no fiable. Se puede extender con plugins, preconfiguraciones personalizadas, y más extensiones.

  • Página Web: https://josm.openstreetmap.de
  • Guía de usuario: https://wiki.openstreetmap.org/wiki/JOSM/Guide
  • Tutorial: https://learnosm.org/en/josm/start-josm/
  • Colección de conectores de JOSM: https://josm.openstreetmap.de/wiki/Plugins

Merkaartor¶

Merkaartor es un editor de mapas de OpenStreetMap que utiliza el kit de herramientas de Qt. Es un poco más fácil de usar que JOSM, pero tiene un poco menos de funciones.

  • Página Web: http://merkaartor.be
  • Ayuda en línea: https://wiki.openstreetmap.org/wiki/Merkaartor/Documentation

QGIS¶

Los datos de OpenStreetMap se pueden cargar en ** QGIS ** como capa vectorial, como una funcionalidad central o a través de plugins como QuickOSM.

  • QuickOSM: https://plugins.qgis.org/plugins/QuickOSM/
  • Ayuda en línea: https://wiki.openstreetmap.org/wiki/QGIS#QGIS_2:_Importing_OSM_vector_layers
  • Tutorial: https://learnosm.org/en/osm-data/osm-in-qgis/

Osmosis¶

Osmosis es un programa muy capaz para realizar muchas tareas a un nivel bruto en datos OSM. Esto incluye datos de importación y exportación a bases de datos, clasificación, limpieza y creación de volcados de datos.

  • Documentación: https://wiki.openstreetmap.org/wiki/Osmosis#Usage
  • Tutorial: https://learnosm.org/en/osm-data/osmosis/

osm2pgsql¶

osm2pgsql es un programa que convierte datos de OpenStreetMap (.osm) en un formato que puede cargarse en PostgreSQL (PostGIS). A menudo se utiliza para renderizar visualmente los datos de OSM utilizando Mapnik, ya que Mapnik puede consultar PostgreSQL para datos del mapa, pero no trabaja directamente con archivos OSM.

  • Página Web: https://wiki.openstreetmap.org/wiki/Osm2pgsql
  • Tutorial: https://learnosm.org/en/osm-data/osm2pgsql/
