Guía de Inicio Rapido de Mapnik¶

Mapnik es un conjunto de herramientas para desarrollar aplicaciones de mapeo. Sobre todo, Mapnik trata sobre la elaboración de mapas hermosos. Es fácilmente extensible y adecuado para el desarrollo web y en escritorio.

Mapnik & Python¶

Mapnik y sus enlaces de python están instalados y listos para ser utilizados para scripting en esta máquina.

Crear mapas en Python es fácil con Mapnik. Abra una ventana de terminal y escriba python en la línea de comandos para introducir un intérprete de Python. A continuación, intente lo siguiente:

>>> import mapnik, os
>>> m = mapnik.Map(600,400)
>>> style = '/usr/local/share/mapnik/demo/population.xml'
>>> mapnik.load_map(m,style)
>>> m.zoom_all()
>>> mapnik.render_to_file(m,'map.png')
>>> os.system('xdg-open map.png')

Nota

El código anterior depende de tener una hoja de estilo XML que Mapnik puede leer , definiendo las capas desde donde se leen los datos y los estilos para aplicar a esas capas. Usted puede crear uno de éstos dentro de QGIS con el complemento de Quantumnik: https://plugins.qgis.org/plugins/quantumnik/

Mapnik & Leaflet¶

Una aplicación de demostración básica está disponible que destaca el uso de Mapnik para servir teselas en un mapa web de OpenLayers en el esquema de teselas OSM/Google.

La demo utiliza un servidor de teselas diseñado para Mapnik llamado «TileStache». Visite el sitio web de TileStache para obtener más información: http://tilestache.org/

  1. Haga click en Desktop ‣ Herramientas espaciales ‣ Iniciar Mapnik & TileStache. El servidor se inicia en una ventana de terminal (que permanece abierta y genera información básica de depuración).
  2. Compruebe que el servidor funciona solicitando un icono del servidor. Haga click este link http://localhost:8012/example/0/0/0.png y observe la salida en la ventana del terminal del servidor.
  3. Visite la página de la aplicación de demostración de Leaflet http://localhost/leaflet-demo.html.
