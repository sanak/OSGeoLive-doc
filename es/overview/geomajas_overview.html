Geomajas¶

Cliente GeoExplorer SIG¶

El Cliente permite a los desarrolladores crear potentes aplicaciones GIS web que incluyen mapas interactivos, widgets extensibles y casos de uso personalizados. Permite el acceso a una amplia gama de fuentes de datos y funcionalidad a través de un navegador web.

Características Principales¶

  • Programable usando Java puro a través de GWT
  • Es compatible con muchos tipos de capas (incluyendo WMS, WFS, WMTS, TMS, OSM, Google Maps, Shape, PostGIS, Oracle, JDBC, …)
  • Práctica colección de widgets integrados (incluyendo Editing and redlining, Print, Geocoding, Layer tree, …)
  • Cree sus propios casos de uso y cree nuevos widgets
  • Fácil de personalizar mediante CSS personalizado
  • Sin necesidad de complementos del navegador
  • Se ejecuta en una amplia gama de dispositivos de Internet con HTML5
  • Características Principales No es un entusiasta de GWT? Acceda a las Características Principales a través de Javascript API

Funciones de Enterprise Server¶

  • Seguridad avanzada en diferentes niveles (capa, función, atributo, funcionalidad, …)
  • Perfecta integración con su propio Modelo de Dominio - persistencia a través de Hibernate, servicios web, Java RMI, …
  • Integración empresarial con otras aplicaciones comerciales tales como JBPM, Activiti, JasperReports, CMS, Alfresco ta través de estándares industriales (SOAP, REST, ESB, JNDI, …)
  • Fácil extensible a través del marco Spring
