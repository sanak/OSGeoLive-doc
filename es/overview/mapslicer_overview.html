MapSlicer¶

Proxy WMS & Servicios de mosaicos cartográficos¶

Publicación de Mapas¶

MapSlicer es una aplicación de escritorio para la creación de map tiles o mosiacos de mapas y la publicación rápida de mapas ráster. Los Geodatos son transformados en teselas compatibles con Google Maps y Earth - listos para publicarse mediante carga directa en cualquier servidor web o almacenamiento en la nube, (por ejemplo Amazon S3).

No es necesaria una configuración extensa en el servidor, cualquier alojamiento simple de archivos está bien. Las interacciones dinámicas, como el paneo y el acercamiento, o la superposición de marcadores y datos vectoriales, son provistas por una poderosa funcionalidad de navegador.

La aplicación genera directamente un visor simple listo para usar basado en OpenLayers y API de Google Maps y se puede personalizar fácilmente.

Características Principales¶

  • Bien conocido Javascript APIs respaldo: OpenLayers, Google Maps API
  • Ninguna instalación de software de servidor adicional necesaria
  • Alojamiento casi por todas partes: nube de almacenamiento como Amazon S3 o cualquier webhosting «ilimitado» y barato con acceso FTP.
  • Mashup sencillo con capas comerciales (Google, Bing, Yahoo) u OpenStreetMap
  • Las teselas se adecúan al Servicio de especificación de mosaicos cartográficos de OSGEO TMS (Especificación del Servicio de Mapas de Mosaico).
  • Capacidad para procesar datos de raster en diferentes formatos: TIFF/GeoTIFF, MrSID, ECW, JPEG2000, Erdas HFA, NOAA BSB, JPEG y más…
