t-rex¶

Vector Tile Service¶

t-rex è un server per tile vettoriali specializzato nel pubblicare tile MVT da dati propri.

Caratteristiche principali¶

  • Supporto per database PostGIS e formati vettoriali GDAL
  • Auto-detection of layers in data source
  • Visualizzatore incorporato per la visualizzazione e l’ispezione dei dati
  • Comando per la generazione delle tile con un parallelizzazione semplice
  • Riproiezione automatica alla griglia CRS
  • Supporto per griglie delle tile personalizzate
