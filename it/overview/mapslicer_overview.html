MapSlicer¶

Servizi di proxy WMS e tile¶

Pubblicazione di mappe¶

MapSlicer è un’applicazione desktop per la creazione di tile di mappe per la pubblicazione rapida di mappe raster. I geodati vengono trasformati in tile compatibili con Google Maps ed Earth - pronti per la pubblicazione tramite caricamento diretto su qualsiasi serve web o spazio cloud (come Amazon S3).

Non è necessaria una configurazione complessa lato server, qualsiasi semplice hosting di file va bene. L’interazione dinamica, come il pan e lo zoom, la sovrapposizione di marker e di dati vettoriali vengono fornite da potenti funzionalità sul browser.

L’applicazione genera direttamente un visualizzatore semplice basato su OpenLayers e sulla API di Google Maps e può essere facilmente personalizzato.

Caratteristiche principali¶

  • Supporto di ben note API in Javascript: OpenLayers e Google Maps
  • Non serve installare software extra su server
  • Hosting possibile quasi ovunque: da spazi cloud come Amazon S3 a qualsiasi web hosting economico con accesso FTP e spazio «illimitato»
  • Facile combinazione con layer commerciali (Google, Bing, Yahoo) od OpenStreetMap
  • Le tile seguono le specifiche OSGeo TMS (Tile Map Service)
  • Possibilità di elaborare dati raster in vari formati: : TIFF/GeoTIFF, MrSID, ECW, JPEG2000, Erdas HFA, NOAA BSB, JPEG e altri…
