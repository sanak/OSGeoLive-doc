zyGrib¶

Previsioni del tempo¶

zyGrib è un programma per scaricare e visualizzare dati di previsioni del tempo nel formato standard di distribuzione GRIB.

Caratteristiche principali¶

  • Visualizzazione di dati meteorologici da file in formato GRIB
  • Scaricamento automatico di previsioni meteomarine
  • Scaricamento automatico di dati IAC (fleet code)
  • Visualizza animazioni di previsioni su otto giorni
  • Create le vostre mappe meteo regionali (per tutto il mondo), o visualizzate una previsione quantitativa per un punto preciso.
  • Traccia grafici a partire da dati di vento, pressione, temperatura, umidità, precipitazione, neve, copertura nuvolosa, punto di rugiada, altezza delle onde e dati di alta quota
