Geomajas¶

Client SIG dans le navigateur¶

Le client permet aux développeurs de créer de puissantes applications Web SIG comprenant des cartes interactives, des widgets extensibles et des cas d’utilisation personnalisés. Il permet d’accéder à un large éventail de sources de données et de fonctionnalités grâce à un navigateur Web.

Caractéristiques principales¶

  • Programmable en utilisant du Java pur via GWT
  • Supporte de nombreux types de couche (incluant WMS, WFS, WMTS, TMS, OSM, Google Maps, Shape, PostGIS, Oracle, JDBC, …)
  • Collection pratique de widgets intégrés (incluant édition et redlining, impression, géocodage, arbre de couches,…)
  • Créez vos propres cas d’utilisation et créez de nouveaux widgets
  • Facile à habiller grâce à du CSS personnalisé
  • Pas besoin de plugins de navigateur
  • Fonctionne sur un large éventail de périphériques Internet utilisant HTML5
  • Pas un fan de GWT? Accédez aux fonctionnalités du cœur via l’API JavaScript

Fonctionnalités du serveur d’entreprise¶

  • Sécurité avancée à différents niveaux (couche, objet géographique, attribut, fonctionnalité, …)
  • Intégration transparente avec votre propre modèle de domaine - persistance via Hibernate, services web, Java RMI,….
  • Intégration d’entreprise avec d’autres applications métier telles que JBPM, Activiti, JasperReports, CMS, Alfresco à travers les standards de l’industrie (SOAP, REST, ESB, JNDI,…)
  • Facilement extensible via le framework Spring
